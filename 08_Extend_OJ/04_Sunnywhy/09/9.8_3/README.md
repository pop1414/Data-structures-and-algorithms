# 最小前缀编码长度

## 题目描述

现需要将一个字符串$s$使用前缀编码的方式编码为01串，使得解码时不会产生混淆。求编码出的01串的最小长度。

## 输入描述

一个仅由大写字母组成、长度不超过100的字符串。

## 输出描述

输出一个整数，表示最小长度。

## 样例

### 样例1

**输入**

```
ABBC
```

**输出**

```
6
```

**样例解释**
将$A$编码为$00$，$B$编码为$1$，$C$编码为$01$，可以得到$ABBC$的前缀编码串$0010101$，此时达到了所有可能情况中的最小长度$6$。

## 解题思路

- 本题是哈夫曼编码，本质与合并果子一样都是WPL
- 最小编码长度就是WPL，每个字符出现的频率就是weight

## 实现要点

- 输入的是字符串，要根据字符串得出每种字符`各自的频率（权重）`
- 通过遍历字符串，然后将频率存入大小26的数组中，`index`为字母的顺序，例如A就是0，`frequnce[0]`就是表示A的权重
- 再将权重存入`min_heap`然后操作
