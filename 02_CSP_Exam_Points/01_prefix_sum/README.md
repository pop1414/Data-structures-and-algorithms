## 知识点速记

- 定义：`sum[i] = a[1] + ... + a[i]`，区间和 `l..r = sum[r] - sum[l-1]`。
- 变种：二维前缀和、差分数组、前缀异或/前缀最大值。
- 常见场景：区间查询、多次求和、统计子数组/子矩阵。

## 适用条件

- 数组是`静态`的，数组元素几乎不可修改（或仅有少量修改，修改次数远少于查询次数）
- 需要`多次区间查询`：比如`反复求区间和/区间内某个值出现的次数/区间异或和`
- 查询的是`可叠加的区间信息`：和、积、异或、计数等（最值类不行，最值需要线段树）。
